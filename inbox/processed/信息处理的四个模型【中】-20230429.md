**模型3：贝叶斯概率模型**

我们前面提到，信息处理是通过“信息存储”和“信息处理”两大行为，将“输入”转化为“输出”。在这个过程的基础上，贝叶斯概率模型是如何运作的呢？

贝叶斯概率模型认为，大脑对世间万物的认识都是由概率组成的。这是一种主观的概率，是我们基于大脑中已存储的生活体验与学习成果，对事物做出的判断。

首先，需要将这里的“主观概率”与客观概率区分开。客观概率展示的是真实的概率，比如一枚均匀的硬币抛在正面的概率本就是50%；而主观概率仅反映了基于当前已有信息能够得到的最佳猜测。

**“先验概率”与“后验概率”**

举个例子，今天早上点点和小华都跟老师说自己的作业忘带过来了。点点的成绩很好，今年95%的作业都完成了；小华则有50%的作业都是没做的。老师依据对她们平时的了解，猜测她们写了作业的概率有多大。

此时可以认为点点写了作业的概率是95%，而小华写了作业的概率是50%。因为这是**基于先前的经验得到的概率，我们暂且称它为****“先验概率”****，记为P(H)**。那么没写作业的概率就记为P(¬H)，P(H) + P(¬H) = 1。

为了进一步了解两个小朋友是不是在撒谎，老师给她们出了一个和作业题有关的小测验。老师认为，写了作业的人有90%的概率能答对这道题，而没写作业的人只有30%的概率能答对。

我们知道H代表「写了作业」，再用E代表「答对题目」。那么写了作业的小朋友答对题目的概率记为P(E|H) = 90%；没写作业的小朋友答对题目的概率记为P(E|¬H) = 30%

最终小华答对了，而点点答错了。

现在有了新的信息作为证据，主观概率也会随之更新。**基于新证据得到的更新后的概率，我们暂且称它为****“后验概率”，记为P(H|E)**。

**贝叶斯公式与“条件概率”**

我们来聊一下关于「贝叶斯公式」我觉得最好的理解方法～

不知道你是否已经发现公式里大写字母的隐喻：

H = Hypothesis，对应着我们进行猜测的事情，也就是假说；

E = Evidence，对应着我们用来更新先验概率的事情，也就是证据。

我们把H和E两种事情分别用黄色和蓝色的形状进行表示。并用绿色表示两者同时发生，标识为B。

如果用面积来表示概率，可以看出来：

        P(H) = H / R；P(E) = E / R.

当H确定发生时，E发生的概率（条件概率）为：

        P(E|H) = B / H.

回到贝叶斯公式，我们如何得到后验概率P(H|E)呢？

        P(H|E) = B / E = (B / R) / (E / R)；

        其中，B / R = (B / H) \* (H / R)；

以上便可以简单地推出贝叶斯公式的形式。

**大脑中的贝叶斯：期望效用**

现在回到我们最初的问题：贝叶斯概率模型是如何将“输入”处理为“输出”的呢？

大脑对于事物的既有认知就是先验概率；当与事物有关的新证据作为“输入”传入大脑时，信息处理机制便将新证据与先验概率结合起来，通过贝叶斯模型来校准对事物的认知，即获得其后验概率作为“输出”。

以上就是贝叶斯概率模型在信息处理机制上的基本概念。由于人对事物的认识并不是非黑即白的，在进行决策的时候也常常摇摆不定，贝叶斯概率模型对此给了一个很好的解释。

这里我们简单聊一聊**“期望效用”**，它是关于**大脑如何处理钱、投资和风险决策**的理论。它在贝叶斯概率估计的基础上，叠加了“效用”的概念。

当我们面对许多种行动方案，并需要从中决策出最佳方案的时候，大脑会试图将我们的“期望效用”最大化。

“期望效用”，可以简单分解为“期望”与“效用”，前者就是我们常说的期望值，而后者对应个体的价值偏好。

“期望”，对应着这些方案可能得到的每一个结果的概率总和；

“效用”，则对应着每一个结果对于我们的价值。

现在来考虑两个选择：

选择1：给你10元钱；

选择2：你从10个红包中抽1个，其中1个有100元，剩下是空的；

选择1的“期望”就是10元钱；选择2的“期望”是100/10，也是10元钱。纯粹从期望的角度来看，两者是相等的。

但如果考虑到“效用”，风险厌恶者可能更愿意选择前者，愿意冒险的人可能更愿意选择后者。同样，由于金钱的边际效用递减，一个不缺10元钱的人可能更愿意冒着丢失10元钱的风险去抽一张红包。

于是，一件事情即使在同样的后验概率上，不同的个体也会由于其“效用”对应的价值偏向不同，而产生不同的决策。

**模型4：动力学模型**

终于到最后一个模型啦～

我们还是从一个小故事开始，蟋蟀是如何辨别声音方向的呢？

蟋蟀的前腿上有一对耳膜用来接收声音。由于声波与两只前腿的距离不同，声波在传输到两只腿的耳膜上时会有时间差；将声音到达的时间延迟作为变量，与声音的方位建立直观计算模型，便可以非常简单地辨别声音方向。

又比如驾驶员在路上开车的时候，车的速度、位置、加速度、路况等变量会随着时间而变化，影响着驾驶员的决策和行为。通过建立这些变量与驾驶员决策行为之间的动力学系统模型，我们可以更好地理解驾驶员的决策和行为过程，并预测他们未来可能的行为。

简单来说，动力学模型认为人的认知过程是一个动力系统，就像太阳系各大行星的运动会有规律地带来季节与温度变化一样，认知系统的演化也可以通过微分或差分的方式进行建模，以此描述人类在动态和不确定环境中做出决策和规划行为的过程。

在动力学模型看来，或许不需要一个庞大的中央大脑来进行系统的行为调节。就像一大群蚂蚁构成一个看似产生了智慧的蚂蚁窝一样，只要每一个组成部分都按照特定的行为方式与其他组成部分交互合作，就能构成一个完整的系统；这也是纳米机器人的灵感。
