这篇文章我们来讲讲大脑的信息处理机制～

通常认为大脑的信息处理就是大脑结合**已存储的信息**，通过**信息处理**，将**输入**转化为**输出**的过程，存在着“信息存储”和“信息处理”两大行为。

本次的主题是关于信息处理机制（即认知过程的底层机制）的四个模型：物理符号系统、神经网络、贝叶斯概率模型、动力学模型。

本文先介绍**物理符号系统**与**神经网络**。

**模型1：物理符号系统**

我们先用一个简单的动作来了解“物理符号系统”。

当你自然而然地拿起桌上的杯子喝水时，你是否想过大脑是如何实现这个过程的呢？直觉上我们可能会认为是经历了下面五个步骤：

找到杯子 -> 触碰杯子 -> 握住杯子 -> 拿到嘴边 -> 喝水

进一步，我们是如何找到杯子，握紧它而不摔碎它，又是如何正确地将它们拿到正确的地方呢？

“物理符号系统”的理论认为，我们的大脑会把我们的生活体验与学习成果抽象成一些概念符号并存储起来。

我们会记得杯子的形状，以至于从任何角度看到它都能认得；我们会从婴儿时期开始学习抓握东西；我们也会掌握空间的距离，知道从手到杯子以及从杯子到嘴边的最短路径。

同样，我们也会把这一整个喝水的过程存储起来；当身体发出喝水的需求时，大脑就会检索到这个过程，并将每一步拆解到更细致的子任务，然后一步一步地完成这些子任务，最终将整个问题解决。

**“信息” & “关于信息的信息”**

了解了基本逻辑以后，我们来详细说说“物理符号系统”的概念。

“物理符号系统”即“由实际符号构成的系统”。在这里，“物理”是指“有实体的”。

“符号”包含基本的**信息**，即实体对象的表征：

“4”，“5”，“猫”；

也包含**关于信息的信息**，即对象的操作规则：

“4”乘“5”等于“20”；饲养猫的步骤12345。

基本的信息就像是一条条的静态知识；而关于信息的信息则是一些描述知识的规则，常常会有像“所有……都是……”或者“如果……那么……”之类的表达。

对于“物理符号系统”，信息存储和信息处理是两个泾渭分明的行为：

**信息存储**方面，大脑将各类“信息”与“关于信息的信息”编码成抽象的概念符号，随着人类的学习与经历，对它们进行创建、修改、复制、删除；

**信息处理**方面，大脑则会基于外界输入的信息，对已存储的“信息”和“关于信息的信息”进行检索并调用，基于过往学到的操作规则来解决问题。

在计算机的应用上，专家系统和一部分机器学习方法（例如决策树、支持向量机等）也都属于物理符号系统的模式，它们将输入与输出的映射关系用公式符号存储在模型中，并将它作为操作规则，应用于新输入的计算、分类与预测。

**“物理符号系统”与语言**

就像上面喝水的例子，当我们试图解释大脑的信息处理机制时，常常第一反应就是“物理符号系统”。为什么我们会非常自然地用“物理符号系统”来解释大脑的底层机制呢？

原因在于，**我们通常会借助语言来进行思考；而语言的底层逻辑就是“物理符号系统”。**

在学习第二门语言的时候应该都有感触，通常的学习方法就是记住“词汇”和“语法规则”，然后在具体的应用中从记忆里调用我们已经学过的内容进行表达。

例如需要用英语表达“昨天我去过车站”的时候，我们会试着想起“yesterday”、“station”，过去时态，以及主谓宾状的语法规则。

语言的逻辑与物理符号系统的逻辑一致，“词汇”对应着“信息”，“语法规则”对应着“关于信息的信息”。于是我们自然而然会相信并习惯这种信息处理模型。

**模型2：神经网络**

神经网络模型的灵感来源于大脑的神经元。相比较“物理符号系统”的直观和容易理解，神经网络则需要小小的绕一个弯子。

你不一定了解神经网络；对于它到底是什么实现的，你也不一定感兴趣。但你一定听说过也用过它，因为如今我们身边充斥着基于各种神经网络搭建的算法应用，包括人脸识别、图像识别、对话机器人等等。

想象一个由很多个神经节点组成的网络，它们连接在一起形成一层一层的形态。上一层节点被激活之后，将刺激传递给下一层。它们对下一层的贡献（权重）有大有小，而下一层接收到的全部刺激如果足够大（阈值），就会被激活并继续向下一层传递。

我们来看一个和二进制加法有关的问题。抽出左边公式的第1位数字，就可以得到右边这个规则：

00 + 00 = 00     =>    0 & 0 = 0

00 + 01 = 01    =>   0 & 1 = 1 

01 + 00 = 01    =>    1 & 0 = 1

01 + 01 = 10    =>   1 & 1 = 0

怎么用神经网络模型来表达右边的信息处理规则呢？

可以使用下图这样的神经网络来进行信息处理（线上的数字对应权重，节点上的数字对应阈值）：最左边的两个输入层分别和箭头上的权重值相乘，并传输到下一层的节点上进行求和。

例如 0 & 1 = 1 这个计算，左边节点输入为0和1：

中间上部的节点： 0 \* 1 + 1 \* (-1) = -1；小于0.5则不激活，为0；

中间下部的节点： 0 \* (-1) + 1 \* 1 = 1；大于0.5则激活，为1；

右边的节点：0 \* 1 + 1 \* 1 = 1；大于0.5则激活，结果为1。

神经网络模型并不会像“物理符号系统”那样，以显性的方式记录规则，而是通过调整权重和阈值来得到并记录正确的处理逻辑。所以我们并不能通过网络直接得到 “如果两个输入相等，输出就是0” 这样容易理解的规则，这也是为什么大家通常称它为黑盒子。

但是另一方面，它又足够简单，只通过“权重”和“阈值”这样的数字，连接后能够表达各种复杂的东西。**在“权重”与“阈值”上，既存储了信息本身，又存储了关于信息的信息；就像我们的基因一样，即存储了基因本身的编码，又存储了能够处理基因表达的蛋白质的编码。**

虽然当下的神经网络模型和我们真正的大脑神经元在很多细节上都并不相似，但神经网络模型依然可以作为一个信息处理的简化模型，来模拟人类的认知活动；至少从chatgpt来看，它对我们的语言能力已经模拟得相当出色了。
