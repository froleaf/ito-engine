在五一长假前，陆奇的演讲笔记几乎席卷了整个互联网。

演讲笔记开头提到的“三位一体结构演化模式”里，包含了三个体系：“信息”系统、“模型”系统、“行动”系统。

“信息”系统对应着如何正确高效地从环境中摄入信息；

“模型”系统对应着如何基于已有信息和操作规则处理摄入信息；

“行动”系统则对应着输出如何与环境进行交互。

它们恰好对应着信息处理机制中的“输入”、“信息存储与处理”、“输出”；放在人类的信息处理框架下，即是感应器、信息处理器、效应器。

而在认知科学所研究的信息处理机制中，弱化了感应器和效应器的部分，将信息处理与信息存储作为主要研究对象，重点研究“信息处理过程”这一环节。

**模型汇总**

在前面两篇文章中，我们介绍了用来解释信息处理机制的四种模型，每一种模型在各自的领域内都是枝叶繁茂、非常庞大的学科。

**物理符号系统**：结合记忆中已有的知识，通过和语言一致的思维方式，用逻辑推演（如IF…ELSE…）进行信息处理；

**神经网络模型**：借鉴神经元激活与发射的模式，将所有信息存储在神经网络的权重和激活值里，通过信息在神经网络中传播或是抑制来处理信息；

**贝叶斯概率模型**：将人对世界的认知表达为先验概率；在新的证据出现后，通过贝叶斯公式将新证据融合进先验概率中，得到后验概率来更新认知；

**动力学模型**：无需中央大脑进行计算，只通过每个组成部分按照特定的行为方式与其他组成部分交互合作，来构成完整的信息处理系统。

物理符号系统更擅长解决高度结构化的、清晰定义的问题；而神经网络则更适合做一些直觉型的判断，比如分类、模拟婴儿行为等。

在解释不同的认知行为时，每一种模型都有其优势和限制。那么如何更大限度地发挥每种模型的优势呢？

**今天我们来介绍一个混合架构：ACT-R模型。**

作为一个认知模型，ACT-R模型中同样包含了感应器、效应器、信息处理和存储四个部分。

之所以称为“混合架构”，是由于它混合了物理符号系统、神经网络和贝叶斯概率模型三种基本模型。

在这之前，我们先讲一讲“陈述性记忆”与“程序性记忆”。

**陈述性记忆与程序性记忆**

**记忆的存储形式**

ACT-R模型将记忆分为陈述性记忆与程序性记忆，对应着不同类型的知识。

陈述性记忆是关于“是什么”的知识，它由一个一个的知识块组成，例如“北京是中国的首都”、“九九乘法表”；程序性记忆是关于“怎么做”的知识，它由一系列的规则组成，例如“如何讲中文”、“如何计算复杂的乘法”。

陈述性记忆中的知识块，以及程序性记忆中的规则，都被认为是通过符号化存储在大脑中的，「物理符号系统」模型就是在这一层面上生效的。

**陈述性****记忆的调用方式**

问题来了：你有没有过这样的体验？生物考试的时候明知道背过这个名词，但怎么也想不起来。

陈述性记忆为什么有时候会失效？ACT-R模型对此给出的解释是，我们记忆中的知识块都有不同的激活级别，激活级别越高，越容易被我们的目标系统检索到。

激活级别与这个知识的质量以及与目标场景的相关性挂钩（非常巧，这和搜索引擎的排序逻辑是一样的）。我们过去使用这个知识的次数越多，说明知识对我们的价值越高质量也就越高；而这份知识与目标关系越大，也就越容易被唤起。

当我们在生活和学习中使用知识的时候，知识的激活级别便会随着我们使用的频率发生变化；在这样的基础之上，当特定目标与知识的相关性达到一定的阈值，便会成功唤起这份陈述性记忆。「神经网络模型」便是在这一层面上生效的。

**程序性****记忆的调用方式**

我们来到最后一个问题：所有规则都有其生效和失效的时候，那么大脑是如何决定“怎么做”的呢？程序性记忆是如何被选择和调用的呢？

ACT-R模型中的“R”，指的就是“理性-rational”，即大脑会基于过去的经验和知识进行最优的选择。所有成为程序性记忆的生产规则，都在过去的经验中有其对应各种场景时达成各种结果（成功与失败）的概率。

上一篇笔记我们聊到了“期望效用”。“期望”指的是在一个选择之下，各种结果发生的概率；而“效用”则是大脑对一个结果的价值判断。

在所有的生产规则（程序性记忆）里，目标系统会根据当下的目标来匹配期望效用最高的规则；而每次行动之后，新的结果会对生产规则的“期望效用”值进行更新。「贝叶斯概率模型」便是在这一层面上生效。

**两类记忆的使用与学习机制总结**

我们可以用表格来描述陈述性记忆和程序性记忆的使用与学习机制。

|  |  |  |  |
| --- | --- | --- | --- |
|  |  | 陈述性记忆 | 程序性记忆 |
| 使用机制 | 符号化 | 知识块 | 生产规则 |
| 亚符号化 | 与知识块相关的激活级别，会影响记忆的检索 | 生产规则相关的“期望效用”值，会影响其是否被选择 |
| 学习机制 | 符号化 | 添加新的知识块到集合中 | 添加新的生产规则到集合中 |
| 亚符号化 | 改变知识块的激活级别，以及块与块之间的关联的强弱 | 改变生产规则的“期望效用”值 |

**ACT-R认知架构**

接下来看看ACT-R认知架构的具体形态。

ACT-R (Adaptive Control of Thought-Rational)，翻译过来大概是“理性思维的自适应控制”，我们主要介绍该模型在作为人类认知模型时，是如何将多个基本模型融合起来的。

ACT-R模型主要包含两个层，分别是“感知-运动层”与“认知层”。

「每个层包含多个模块，层与层之间的交互通过各自的缓冲区进行，因此缓冲区也被称为工作区。缓冲区非常重要，但不是我们今天的重点，所以我们不用纠结于缓冲区的功能。」

（图翻译自José Luis Bermúdez 的《Cognitive Science》教材第三版）

感知-运动层作为和外界环境进行交互的模块，分为以下两个部分：

感知模块：包含听觉模块、视觉模块等进行感官输入的模块；

运动模块：包含语言模块、手工模块等通过行为进行输出的模块。

认知层则由陈述性记忆、程序性记忆、目标系统（进行模式匹配与生产执行）构成。陈述性记忆和程序性记忆对应着两种不同的知识，而目标系统则是决策发生的模块。

当感知-运动层摄入信息后，信息通过缓冲区到达认知层的目标系统；目标系统匹配合适的程序性记忆，并唤起需要的陈述性记忆，产生决策与执行动作，最终通过缓冲区返回给感知-运动层，以进行与外界的交互。
