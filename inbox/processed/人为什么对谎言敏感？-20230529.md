进化论认为，人类需要与他人合作，以提高生存和繁衍后代的机会。

然而，社会互动中也存在着很多欺骗行为。

因此，人类需要能够识别他人是否在说谎，以在社会生活中获得优势。

今天想和大家分享的「说谎检测模块」，就是关于人类在识别说谎这方面的专业化认知能力。

在正式介绍之前，我们先来玩一个游戏～

**我们对谎言的敏感性**

假设我们随机拿到四张卡片：这些卡片一面是字母，另一面是数字。

规则是“如果一张卡片的正面是元音，那它的反面就应该是偶数”。

但是并不是每一张都符合这个规则。

现在四张卡片单面朝上，像这样放在你的面前。

你要做的事情是，翻开卡片，来确定每一张卡片是否符合规则。

那么我们开始吧！

第一张需要翻开吗？显然需要，如果反面是偶数才符合规则；

第二张需要翻开吗？显然不需要，无论反面是什么数字都符合规则。

第三张需要翻开吗？第四张呢？

如果不稍微深入地进行一下逻辑思考的话，大概率会搞错。

你可能会想要翻开第三张，看看反面是不是一个元音；而实际上反面无论是元音还是辅音，这张卡片都是符合规则的。

第四张看起来似乎没有什么必要翻开；但其实是需要翻开的，因为如果反面是一个元音，这张卡片是不符合规则的。

游戏并不复杂，你也许会觉得“这有什么呢，虽然第一反应可能有误，但我稍作思考就能给出正确答案。”

那我们就来看看另一个游戏。

随机拿到四张卡片：这些卡片一面是关于饮品的文字，另一面是关于年龄的数字。

“如果一张卡片的正面写着「酒」，那它的反面就应该大于「18」岁”。但是并不是每一张都符合这个规则。

现在四张卡片单面朝上，像这样放在你的面前。

**你是一个警察****，**你要做的事情是翻开卡片，确定每张卡片是否符合规则。

你要翻开哪几张卡片呢？

你应该可以不假思索地确定，要查写着「酒」的卡片和数字小于「18」的卡片，第一张和第四张。

两个游戏的模式甚至卡片顺序都是一样，为什么对我们来说难度却完全不同呢？背后就是认知系统的「说谎检测模块」在发挥作用。

「说谎检测模块」是认知科学中「达尔文模块假说」的一部分。

达尔文模块假说指出，**人在道义条件下的推理，比非道义条件的普通推理要擅长得多，这正是由于道义条件激活了**认知系统中的“说谎检测模块”**。**

**道义对应的英文单词是“deontic”，通常表达人们应该或不应该做什么，例如“你应该说实话”或“你不能偷东西”。**

第一道题是普通的逻辑推理题，而第二道题把你假定为“警察”，这个时候便触发了人的“说谎检测模块”：你喝酒的话，就应该超过18岁，否则我会知道你在说谎。

因此，相比于第一个游戏，我们会更擅长第二个游戏。

**达尔文模块与说谎检测**

接下来，我们就来宽泛地了解一下达尔文模块的内容。

达尔文模块假说认为，人类的认知与思维能力是进化的产物。

在祖先们生存生产与合作的过程中，通过进化和自然选择形成了专业化的认知模块，用于解决祖先们遇到的诸如社会协调之类的特定问题。

例如面部识别、情绪检测、注视跟随、心理学常识（理解和预测他人行为与心理状态）、物理学常识（理解物体的整体性、区分接触面、理解物体运动的连续性等）、生物学常识等能力，都是各种专业化模块赋予的。

而人对权利与义务、规则与权限、许可与禁止等概念的敏感性，也对应着一个专业化模块，即上面提到的“说谎检测模块”。

这是一个与社会活动息息相关的能力。

从进化的角度而言，**说谎检测模块可能是人类产生「合作」的基础，是「利他行为」出现的一种解释。**

那为什么「说谎检测模块」会在自然选择的过程中被保留下来？

人类的合作与利他行为又是如何在说谎检测模块的支持中诞生的？

接下来，我们来展开讨论关于“合作与背叛”的场景。

**囚徒困境**

博弈论中的「囚徒困境」大家也许都耳熟能详了。

有两个犯人被分开审讯。警长确认了他们的罪行，但苦于缺少证据，所以提出这样的方案：

如果两个犯人都承认罪行，那么每个人坐5年牢；如果只有一个犯人认罪，那么认罪者直接释放，而另一个人坐10年牢；如果没有犯人承认罪行，那么每个人坐2年牢。

|  |  |  |  |
| --- | --- | --- | --- |
|  |  | 犯人2 | |
|  |  | 认罪 | 不认罪 |
| 犯人1 | 认罪 | -5，-5 | 0，-10 |
| 不认罪 | -10，0 | -2，-2 |

在这个规则下，认罪者也就是背叛者。

从全局来看，两个人都互不背叛，各坐2年牢是最好的结果。

但如果从个人角度来看，无论对方选择背叛还是不背叛，犯人本人选择「背叛」都会是最好的结果。

对于犯人1来说：

如果犯人2认罪，那么犯人1认罪会比不认罪减少5年的牢狱之灾；

如果犯人2不认罪，那么犯人1认罪就可以直接释放。

在囚徒困境中，从利己的角度出发，似乎永远得不出合作的选项。

那么利他行为是如何出现的呢？

**无限迭代的囚徒困境**

我们把囚徒困境变成一个更一般的场景。

两个玩家进行积分游戏。玩家可以选择背叛或者合作，表格里是对应动作能够得到的积分。

|  |  |  |  |
| --- | --- | --- | --- |
|  |  | 玩家2 | |
|  |  | 背叛 | 合作 |
| 玩家1 | 背叛 | 0，0 | 5，-5 |
| 合作 | -5，5 | 3，3 |

和囚徒困境一样：双方合作是全局最优选项；而对于个人来说，背叛永远是最好的结果。

如果这是一个在两位玩家之间只玩一盘的游戏，从个人利益最大化的角度考虑，自然应该选择背叛。

**那么，如果这是一个在多位玩家之间随机进行的多轮游戏呢？**

假设好人总是选择合作，而坏人总是选择背叛。

在这样无限迭代的囚徒困境中，一个老好人在遇到好人时得3分、遇到坏人时扣5分；一个坏人在遇到好人时得5分，遇到坏人时不得分。

如果玩家大多是好人，那但凡出现一个坏人就会占尽便宜。

而如果玩家大多是坏人，那谁也得不到一点好处。

为了能够得到更高的分数，此时最佳解法是「针锋相对」。

**针锋相对与说谎检测**

「针锋相对」的策略是这样的：首次选择合作，其后每次都使用对方前一次的选择。

这样，他总能够在少量的损失后，找到和一个玩家长期对局时的最优解。

遇到好人会达成共赢选项得3分；遇到坏人则在第一次失去5分后就不再合作；而遇到同样采用「针锋相对」策略的人，也会达成共赢的3分选项。

针锋相对的策略让游戏者只在每个坏人面前吃亏一次，同时与所有不是坏人的人达成长期的友好合作关系。

但实际生活中，对方是否选择了背叛，并不像游戏中这么显而易见。

**想要正确地实施针锋相对的策略，就需要具备正确识别背叛者的能力**；而背叛者常常通过欺骗的方式隐瞒自己的行为。

这就回到了我们在前面提到的，人类在进化过程中具备的专业能力模块：“说谎检测模块”。

**在说谎检测方面具备更优能力的人，才能够在现实中更好地采用「针锋相对」策略，借此通过合作达成共赢的局面。**

这种保护人类祖先在寻求合作的同时不被伤害的能力，通过自然选择被一代代保留下来，成为了我们的一部分。
