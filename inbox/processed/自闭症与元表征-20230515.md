在如今社交媒体的宣传下，很多人认知中的“自闭症”是盛产天才的群体。或许他们更多地被称为“阿斯伯格综合症”，被形容成是一群具有某些特殊才能，但在生活自理和社交能力上存在缺陷的人。

然而“阿斯伯格综合症”与常说的“自闭症”其实有着显著的差异。

自闭症的研究起源于那些基本缺失社交能力、只会模仿和重复别人的话而不会说完整的句子、感官异于常人、强迫症般地遵守自己的流程、甚至常有自残倾向，完全无法正常生活的低龄儿童；他们的智力也未必有突出表现。

“三重损伤”是比较典型的自闭症框架理论。它认为对自闭症有决定性影响的核心因素包含三个点：

1. 常规社交技能方面的损伤；

2. 包括肢体语言在内的语言技能的损伤；

3. 无法进行“社交想象”，例如无法进行角色扮演这类游戏。

阿斯伯格综合症概念的出现则相对晚一些。

阿斯伯格是一位奥地利儿科医生。他在二战时接触了一批患有自闭症的孩子，但他选择接触的孩子是那些智力超群、语言能力强，仅在阅读他人情绪和意图上存在障碍的人。这也是“阿斯伯格综合症”名称的起源。

虽然两者同属于自闭症谱系障碍，但是两个人群的特征相去甚远。

今天我们要聊的是在认知科学角度下，“自闭症”儿童与普通孩子的区别，并由此延伸出人类的一大认知能力：元表征。

**元表征与**扮演游戏****

**表征（representation）通常被认为是人类非常重要的抽象功能。**

例如在路边看到小白狗的时候，我们能够轻易地采用“狗”这个词来表征它，用“白色”来表征我们所看到的颜色，用“小”来表征我们所判断的它的年龄。

于是“小白狗”这个表征成为实体的抽象概念被存在大脑中。

**而元表征，则是“关于表征的表征”。**

我们回到刚才的例子，我将自己在路边遇见的狗表征为“小白狗”。而这时我突然决定用“哮天犬”这个词来表征“小白狗”这个概念。

在这里，“哮天犬”即是关于表征的表征；我知道它只是小白狗，但我又故意把它视为神话故事中的角色，同时可能会开始寻找适合的二郎神。

看起来是一件很幼稚的事情，没错。这只是一些简单的扮演游戏，在我们刚满一岁左右就开始反复进行下列各种扮演游戏：

首先是通过自身发起的扮演游戏：用空杯子假装在喝水；

然后是与外物相关的扮演游戏：替洋娃娃或奥特曼玩具配音；

将对象替换为其他物品：用香蕉当电话的话筒玩过家家；

凭空想象对象：通过想象创造一个想象中的朋友。

这些游戏都在帮助小孩锻炼他们的元表征能力。

**然而人们却发现，自闭症儿童基本不进行扮演游戏。**

**关于自闭症儿童的实验**

自闭症儿童不进行扮演游戏，是否是由于其元表征能力的缺失呢？为了了解这一点，一些心理学家与认知科学家进行了对照实验。

他们把孩子分成三个组。

第一组是6到16岁的自闭症儿童，平均年龄11岁11个月；

第二组是7到17岁的唐氏综合征儿童，平均年龄10岁；

第三组是3到6岁的正常发育儿童，平均年龄4岁5个月。

先对三组儿童进行了口语测试（将词汇与对应的图片连接起来）和非口语测试（记忆测试）。结果表明，自闭症组的儿童口语能力和非口语能力都是显著高于正常发育组的。

|  |  |  |
| --- | --- | --- |
|  | 平均口语  能力年龄 | 平均非口语  能力年龄 |
| 自闭症组 | 5岁5个月 | 9岁3个月 |
| 唐氏综合征组 | 2岁11个月 | 5岁1个月 |
| 正常发育组 | 4岁5个月 | 4岁5个月 |

接下来他们进行了与元表征相关的理解力测试。

测试是用电视播放了这样一段剧情：

小孩A把球放进了篮子里，然后离开了房间。之后小孩B趁A不在的时候，把球从篮子里拿出来，放进纸盒里。然后A回来了。

提问：A认为球在哪里？他会去篮子里还是纸盒里找球？

测试员通过记忆测试和剧情测试，确保所有小朋友都完全记住并理解了剧情内容。接下来就是统计每一组小朋友给出的回答。

唐氏综合征组回答正确的比例是86%；正常发育组回答正确的比例是85%；而自闭症组回答正确的比例仅有20%。

自闭症组的小朋友显著地缺乏这种“元表征”的能力。

**元表征的习得：TOMM模型**

你也许会感到好奇：这种所谓的元表征能力，这种从一岁就开始练习扮演游戏所带来的能力，有什么用呢？

**在认知科学的解释中，元表征赋予我们理解他人的能力，即共情的能力。**

我们通过TOMM（Theory of mind mechanism）模型来解释元表征的作用。TOMM翻译过来是心智机制理论，它将儿童在观察他人方面的心智发展分为三个阶段。

（图翻译自José Luis Bermúdez 的《Cognitive Science》教材第三版）

第一阶段在婴儿9个月及以前。这个阶段内婴儿会掌握一些基本的检测能力。

意向检测：例如能够意识到某个人正在向某处走去；

情绪检测：例如能够察觉到某个人开心或生气；

眼方向检测：能够通过一个人的眼睛判断他看向的地方。

意向检测与眼方向检测都是二元检测能力：例如“A向B走去”；“C看向D”等。

第二阶段出现在9-14个月。这个阶段的婴儿开始掌握共享注意力机制，这也是最初级的元表征能力。

例如：爸爸看到我在喝水；我听到妈妈在打电话。

与第一阶段不同，共享注意力机制是一种三元检测能力，它使婴儿能够处理类似于“A看到B在做C事”这样的复杂意识，同时也开始发展语言能力。**自闭症儿童也是从这个阶段开始异于其他孩子。**

第三个阶段则包含了共情能力的出现与社会性发展。

在这个阶段，孩子们通过进行扮演游戏来深化元表征的能力。他们的元表征能力从初级的三元检测逐渐发展完整。

从“爸爸看到我在喝水”到“爸爸看到我在假装喝水”，他们开始理解世界上有虚构的现实，有错误的信念。

由此，孩子掌握了代入和理解他人的行动及内心世界的能力。而这种复杂的认知能力又能够进一步帮助他们获得语言逻辑的发展。

也因此，没有办法正常获得这种能力的自闭症小孩群体，在语言能力和社交能力上都出现了明显的缺陷，融入正常的社会生活变得遥不可及。

**写在最后**

为了写这篇文章，我专门读完了《不同的音调：自闭症的故事》这本书来了解自闭症的发展历史。

最初我在写框架的时候，是准备用“自闭症儿童占比”之类的开头来做引子的，但是在了解其历史的时候才知道，自闭症的诊断在各历史时期都并没有统一标准，以至于所谓占比从1/500到1/38的版本都有；尤其当延伸到阿斯伯格综合症这类可以在社会上进行正常生活的广义的自闭症谱系障碍群体的诊断标准时，会更加地含糊不清。

当然我也是第一次清楚地意识到，阿斯伯格综合症与自闭症的差距这么大。

想要分享书里的一个故事给大家。

与自闭症不同，阿斯伯格高智商患者中，有许多有天赋、聪明又极有口才的活动家们。他们为自己的自闭特质感到骄傲，发起活动宣称自闭症只是一种“神经多样性”，和LGBT一样不应该被治疗、应该被尊重；他们反对一切针对自闭症的医疗研究。

> 有越多像尼尔曼这样有天赋、聪明又极有口才的活动家们宣称为自己的自闭特质骄傲，就有越多人会忽视那些被自闭症病情严重损伤了正常功能的人。她关心的是那些永远无法开口说话的、需要时刻受到照料以免半夜梦游到河里或泳池中溺亡的以及成年后每天依然需要至少换两次尿布的人。这些患者难以接受采访，因此，他们的故事很少能在新闻上播放。这让公众无法全面理解自闭症带来的损害，也不知道它究竟会剥夺多少生者的机会，更不了解患上“真正的自闭症”（一些家庭为了对神经多样性运动表示轻蔑，在私下使用的说法）的人是什么样子。
>
> 不同的音调：自闭症的故事

自闭症儿童的母亲试图与这位聪明的活动家沟通，但完全没有用。作为自闭症谱系障碍中的一员，“他信念坚定，而且反映出他无法理解他人的观点。”

很难形容我阅读这段内容时的感受。

也许对阿斯伯格综合症的关注，能够吸引更多的人去了解自闭症这个群体。

又或许，这会让自闭症痛苦的一面被阿斯伯格光鲜的外表所掩盖。
