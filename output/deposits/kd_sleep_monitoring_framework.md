## 导读

用户在工作中整理的睡眠监测技术全栈文档，分为两大部分：

**一、睡眠监测技术栈**

三层监测指标体系：
- **临床金标准 PSG**：多导睡眠图，综合 EEG（脑电）、EOG（眼动）、EMG（肌电）、ECG（心电）、呼吸/血氧等通道，精确划分 NREM 1-3 期和 REM 期。EEG 核心：α波→θ波（N1过渡）、纺锤波+K-复合波（N2占50%+）、δ波（N3深睡恢复+免疫）、低幅θ+β波（REM类似清醒但肌张力消失）。
- **可穿戴设备 PPG**：绿光 LED 照射皮肤检测血流变化→提取心率(HR)、心率变异性(HRV)、血氧(SpO₂)，结合加速度计(体动)，通过随机森林/SVM/LSTM 等机器学习模型分类睡眠阶段。局限：REM 与清醒心率特征相似易误判、信号受皮肤/佩戴/温度影响、个体差异大。
- **非接触式技术**：压电薄膜床垫、毫米波/UWB 雷达、红外摄像头+AI、Wi-Fi 人体微动感知（MIT RF-Sleep）等。

**二、四维睡眠健康评估框架**（核心工作产出）

将睡眠健康拆解为四个可量化的维度，每个维度含数据指标层+干预建议层：

1. **入睡健康度**：入睡时间点、潜伏时长、入睡前心率/体温下降曲线。干预：生物钟同步（光照锚点法）、睡前行为优化（数字戒断/体温调节/自主神经平衡/饮食禁忌）、CBT-I 认知行为疗法。
2. **睡眠时长**：日/周时长达标性、睡眠分布、日间休息。干预：睡眠银行管理（NASA小睡法/周末补偿/慢性不足恢复阈值）、年龄/基因型适配。
3. **睡眠质量**：N3深睡+REM时长、周期完整性、夜间HRV/心率/体温。干预：环境工程学（声学/光学/触觉）、生理增强（运动类型与时序/营养时序/压力管理）。
4. **睡眠规律性**：入睡/起床/中值/时长的日间变异性。干预：社会时差量化、相位重置技术（强光疗法/褪黑素时序/TRF）、跨时区/倒班预案。

这个四维框架的设计逻辑与 td_medicine_3_0 中"健康数据四维度"的思路一脉相承——从可测量指标出发，每个维度都有"数据层→诊断层→干预层"的完整闭环。

## 原文

睡眠模拟算法
1. 睡眠识别指标：
  1. 多导睡眠图（PSG）：过多通道传感器连续采集整夜睡眠数据，用于监测睡眠期间的各项生理指标，比如脑电、呼吸、血氧等。特别地，它是诊断睡眠呼吸暂停综合征的黄金标准
    - 脑电图（EEG）‌：监测脑电波活动，区分睡眠阶段（如浅睡、深睡、REM期）‌；
    - 眼动电图（EOG）‌：记录眼球运动，辅助识别REM睡眠；
    - 肌电图（EMG）‌：检测肌肉活动（如下颌、腿部），用于识别异态睡眠（如磨牙、不宁腿综合征）‌；监测肌肉活动，区分REM和深度睡眠；
    - 呼吸相关指标‌：包括口鼻气流、胸腹运动、血氧饱和度（SpO₂）等，用于诊断呼吸暂停、低通气事件；
    - 心率变异性（HRV）：反映自主神经系统状态，与睡眠阶段相关；
    - 其他参数‌：心电图（ECG）、体位变化等。
  2. 行为与环境指标：
    - 体动监测：通过加速度计（如手环）判断卧床和翻身频率。
    - 压力分布：床垫传感器检测身体压力分布，判断是否卧床。
    - 环境参数：光线、温度、声音对睡眠质量的影响。（地理位置与温度、湿度、自然光相关）
  3. 间接指标：
    - 心率与呼吸节律：通过毫米波雷达或摄像头非接触式监测。
    - 皮肤电活动（EDA）：反映交感神经兴奋程度。
    - 体温变化：核心体温下降常伴随入睡。

2. 指标监测方法
  1. 临床金标准：
    - 多导睡眠监测（PSG）：综合EEG、EOG、EMG、ECG等信号，精确划分睡眠阶段，但需实验室环境，成本高。
  2. 可穿戴设备：
    - 智能手环/手表：通过加速度计（体动）和PPG（光电容积脉搏波）估计睡眠阶段（浅睡、深睡、REM）。
      - PPG（光电容积脉搏波）：手环/手表背面的 绿光LED 照射皮肤（如手腕），血液中的血红蛋白会吸收光线。当心脏收缩时，血管内血流量增加，吸收的光线更多；心脏舒张时，血流量减少，反射的光线更强。
        - 心率（HR）：通过PPG波形的周期计算。
        - 心率变异性（HRV）：通过PPG波形的微小时间间隔变化分析自主神经系统状态。
        - 血氧饱和度（SpO₂）：通过红光和红外光吸收比例计算（部分高端设备支持）。
    - 头戴式设备：如Dreem头环，直接采集EEG信号，适合家用场景。
  3. 非接触式技术：
    - 压电薄膜床垫：通过压力变化检测呼吸和体动。
    - 毫米波雷达/UWB雷达：利用电磁波反射信号分析呼吸、心跳和体动（如SleepScore Max）。
    - 摄像头+AI视觉分析：通过红外摄像头监测体动和呼吸（需隐私保护）。
  4. 环境传感器：
    - 智能床垫/枕头：内置压力传感器和光纤传感技术。
    - 声呐/超声波：通过声波反射监测身体微动。（基于Wi-Fi/蓝牙信号的人体微动感知（如MIT的RF-Sleep）。）
经典脑电波算法
一、 EEG对睡眠分期的核心作用
1. NREM睡眠（非快速眼动期）
  - NREM 1期（浅睡期）‌：EEG显示α波（清醒状态）逐渐减弱，θ波（4-7Hz）开始出现，脑电活动趋于平缓，标志着从清醒到睡眠的过渡。
  - NREM 2期（轻度睡眠期）‌：特征性表现为睡眠纺锤波‌（短暂爆发的12-14Hz脑电波）和K-复合波‌（高幅慢波后跟随低幅快波），此阶段占成人总睡眠时间的50%以上。
  - NREM 3期（深睡期/慢波睡眠）‌：EEG以高幅δ波（0.5-4Hz）为主，占比超过20%，此阶段与体力恢复、免疫功能调节密切相关。
2. REM睡眠（快速眼动期）
  - EEG呈现低幅θ波（4-7Hz）和β波（12-30Hz），波形与清醒状态类似，但伴随快速眼球运动（EOG监测）及骨骼肌张力消失（EMG监测），此阶段与梦境、记忆巩固相关。
二、 EEG异常与睡眠障碍的关联
- REM期异常‌：若EEG显示REM睡眠在入睡后几分钟内提前出现（正常需90分钟），可能提示发作性睡病或嗜睡症。例如，患者REM期脑电活动紊乱且频繁觉醒，可结合多次小睡潜伏期试验（MSLT）确诊。
- 深睡眠不足‌：NREM 3期δ波占比显著降低，可能由失眠、睡眠呼吸暂停或神经系统疾病导致，需结合呼吸、血氧等参数综合判断。
三、 EEG与其他监测参数的协同分析
PSG通过同步记录EEG、EOG（眼动）、EMG（肌电）等参数，避免单一指标的误判。例如：
- REM期行为障碍：EEG显示REM期脑电特征，但EMG显示下颌或肢体异常活动，提示患者可能出现梦境演绎行为（如拳打脚踢）‌。
- 微觉醒事件：EEG突然出现α波或高频活动（持续≥3秒），可能由呼吸暂停或肢体抽动引发，导致睡眠片段化

如何用PPG和加速度计估计睡眠阶段
智能手环/手表通过 多传感器融合（PPG + 加速度计）和 机器学习算法 实现睡眠阶段的分类（清醒、浅睡、深睡、REM）。以下是具体流程：
（1）数据采集
- PPG信号：持续监测心率和血流变化（每秒多次采样）。
- 加速度计：监测体动（如翻身、手臂摆动），采样频率通常为10-50Hz。
（2）特征提取
从原始信号中提取与睡眠相关的 时域、频域特征：
- PPG信号特征：
  - 心率（HR）的均值、方差、动态变化（如深睡时心率变慢且稳定）。
  - 心率变异性（HRV）的频域指标（如低频/高频功率比，反映交感/副交感神经平衡）。
  - 血流搏动幅度（REM阶段可能因梦境导致血流波动）。
- 加速度计特征：
  - 体动频率和幅度（清醒或浅睡时体动较多，深睡时体动极少）。
  - 静止时长占比（连续静止时间越长，可能处于深睡或REM阶段）。
（3）睡眠阶段分类算法
通过机器学习模型将特征映射到睡眠阶段：
- 经典方法：
使用 随机森林、支持向量机（SVM） 等分类器，基于历史标注数据（如PSG金标准）训练模型。
例如：若某时段心率低、HRV低频占比高、体动极少，则判定为 深睡。
- 深度学习方法：
使用 LSTM（长短期记忆网络） 或 1D-CNN 处理时序信号，捕捉心率与体动的动态关联。
（4）实际应用中的优化
- 噪声处理：
  - 运动伪影（如夜间翻身）会导致PPG信号失真，需通过加速度计数据识别并剔除噪声段。
  - 使用信号滤波（如小波去噪）提升PPG质量。
- 个性化校准：
根据用户基线数据（如日常静息心率）调整模型阈值。
 智能手环的局限性
- PPG的精度限制：
  - 信号易受皮肤颜色、佩戴松紧、环境温度影响。
  - REM阶段与清醒阶段的心率特征相似，易误判（需结合体动数据辅助区分）。
- 算法依赖标注数据：
多数消费级设备的模型基于实验室PSG数据训练，但个体差异可能导致偏差。
典型案例
- Apple Watch：
结合PPG和加速度计，利用《Sleep Stage》算法（基于PSG数据训练）提供睡眠阶段报告。
- Fitbit：
使用"PurePulse"PPG技术，重点分析心率和HRV，结合体动判断REM睡眠。
