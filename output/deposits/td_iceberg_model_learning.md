# 冰山模型与学习方法论——术源于道

> **类型**：思考沉淀
> **状态**：沉淀
> **涉及领域**：系统论、认知科学、教育
> **关联节点**：系统思维、学习与记忆、元认知与学习科学
> **更新时间**：2023-01-19

## 导读

用户在2023年1月的自我探索期间，重新发现了麦克利兰（McClelland）1973年提出的冰山模型，并将其与自己的学习方法论深度融合。

冰山模型的四层：
1. **Events（事件）** → 观察冰山之上发生的事情 → 做出反应
2. **Patterns/Trends（模式）** → 探索事情发生的模式 → 具有预测能力
3. **Structures（结构和系统）** → 关注结构 → 具备设计系统的能力
4. **Mental Models（心理模式）** → 设计系统者的心理模式 → 获得真正改变的能力

核心洞察：用户一直坚持读源头教科书而非速成笔记/项目经验，现在用冰山模型解释了这个偏好——"目前所有的速成版材料能够给到人的都仅仅是这最浅的两层能力，都是术。当人开始关注结构和系统，便具备了设计系统的能力；而最底层则是设计系统者的心理模式，并真正获得改变的能力。"

最终提炼为一句话：**"术源于道，且枝叶繁茂永远也学不尽；道才是根，是一切事物融会贯通的根本。"**

这个思考同时连接了NP问题类比（01.16）：知识让NP完全问题降级为普通NP问题，求解时间复杂度呈对数下降。知识不是让"看起来简单的成功"变得简单，而是让"不可能"变为"可能"。

## 原文

2023.01.19

今天还有一个困扰也突然被解开。我一直相信学东西要去找源头，要看教科书或者最初始的论文；不能习惯于看别人的项目经验、读书笔记或者速成书与文章。但是我没法解释为什么我觉得这样是对的，一直以来都只能说成是我的直觉或者偏好，也无法说服其他人认可这个想法。但是现在我觉得可以解释了，解释的方式就是冰山模型。冰山模型是美国心理学家麦克利兰于1973年提出的，最初版本由4个部分组成（后面被各种人改成各种版本），从上到下分别是Events, Patterns/Trends, Structures, Mental modes。当人只去观察冰山之上发生的【事情】，就会在事情发生的时候【作出反应】；当人开始探索事情发生的【模式】，便开始具有【预测】的能力。目前所有的速成版材料能够给到人的都仅仅是这最浅的两层能力，都是术。当人开始关注【结构和系统】，便具备了【设计】系统的能力；而最底层则是设计系统者的【心理模式】，并真正获得【改变】的能力。去探究事情因何而发生，找寻世界的本原，这才是道。术源于道，且枝叶繁茂永远也学不尽；道才是根，是一切事物融会贯通的根本。
