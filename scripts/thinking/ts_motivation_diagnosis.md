# 动机与行动力诊断法

> **适用场景**：拖延时自检、评估是否值得投入、理解他人的动力缺失
> **来源**：td_motivation_five_sources（动机来源五维框架）+ td_procrastination_dynamics（拖延动因动力学模型）+ ps_grit_theory（坚毅理论）
> **调用证据**：暂无 appliedIn 记录，来源为用户 2023.05 和 2025.06 的原创框架
> **编译时间**：2026-03-10

## 步骤

1. **动机五维扫描**：对当前事项逐一检查——
   - 内在需求：我真的需要/想要这个吗？
   - 个人优势：这是我擅长的方向吗？
   - 社会认可：做这件事能得到认可吗？
   - 有用：这件事对我或他人有实际用处吗？
   - 价值观：这件事符合我的信念吗？

2. **核心三维聚焦**：需求 + 有用 + 价值观是最重要的三个动机源。如果这三个都不亮，社会认可和个人优势也救不了。

3. **如果动机存在但行动停滞——切换到力学分析**：
   - **驱动力四象限**：
     - 自我后推（问题驱动）/ 自我前拉（目标驱动）
     - 他人后推（期待驱动）/ 他人前拉（从众驱动）
   - **阻碍力两维**：
     - 主观：生理精神健康程度（是不是单纯累了？）
     - 客观：事情难度与卡点（具体卡在哪一步？）

4. **找"钩子"**：能改变静态现状的最小干预点
   - 作用于驱动力：找一个更清晰的目标？找一个同伴？
   - 作用于阻碍力：降低第一步难度？先休息恢复？

5. **Grit 校验**：这件事是"指南针"（长期方向一致性）还是"烟花"（短期兴奋）？
   - 如果是指南针——值得在阻碍力上投入更多能量
   - 如果是烟花——允许自己放下，不是每件事都值得坚持
   - 记住：时间窗口越长，Grit 的影响越大

## 你的独特视角

你自己诊断过"不缺发散、思考、框架搭建、类比和逻辑能力，但缺乏聚焦和落地能力"。根源是损失厌恶——害怕损失机会成本让什么都想抓，害怕损失沉没成本让不敢真正投入。这个脚本的价值不在于分析别人，而在于对自己诚实：动机五维中哪些是真的亮着的，哪些只是你以为亮着的。
